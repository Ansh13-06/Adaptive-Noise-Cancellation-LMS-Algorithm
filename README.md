# Adaptive Noise Cancellation (ANC) using LMS

This project implements an **Adaptive Noise Cancellation** system utilizing the **Least Mean Squares (LMS)** algorithm. It is designed to filter out unwanted background noise from a primary signal by adaptively updating filter coefficients.

## 🚀 Project Structure

As seen in the project architecture (image_2cae37.png):

- `filters/`: Contains `lms_filters.py` for the core LMS logic.
- `models/`: Contains `adaptive_neuron.py` for model architecture.
- `signals/`: Logic for synthetic or real signal generation.
- `visualization/`: Utilities for plotting original vs. cleaned signals.
- `main.py`: The entry point to run the simulation.
